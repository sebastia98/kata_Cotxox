Información general

La aplicación fue desarrollada a partir de una simple idea planteada por David Gelpi, profesor de programación en el instituto C.I.F.P Francesc de Borja Moll. La aplicación debía consistir en el ofrecimiento de un servicio completo entre el transportista y el cliente.

Dejó de ser una idea y se convirtió en un proyecto en desarrollo el 7 de noviembre del 2019 gracias a Sebastià Adrover, desarrollador de software cursando un grado de Desarrollo de Aplicaciones Web de modalidad dual y alumno de David, y el 12 de diciempre salió al mercado su primera y única versión funcional.

Desarrollo

La lógica de la aplicación está totalmente desarrollada con Java y usa Spring como framework para poder controlar las peticiones web del cliente y conectarse a una base de datos MongoDB. Tratar con documentos .json resulta mucho más práctico para establecer un mejor acceso de nuestro servidor a la base de datos sin estropear la modulación del código y manteniendo su seguridad. Ese es el motivo por el cual esta aplicación ha demostrado ser rápida, fiable y eficaz en todos los sentidos convirtiéndose en un software con potencial para ser el mejor del mercado.
